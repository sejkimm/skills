# Learning Material Route

## Use This Route When

The user wants a distilled, reusable learning note: a concept explanation, mechanism, technical note, product or platform analysis, paper note, troubleshooting note, comparison, or durable professional-learning artifact.

## Do Not Use This Route When

- The user wants a raw clipping or source archive.
- The item is mainly course progress, a quiz, or a certification artifact.
- The item is tied to one job, interview, application, or admissions process.
- The item only makes sense inside one project.
- The user wants a reusable template rather than a note.

## Required References

Read these files in order:

1. `references/routes/learning-material/taxonomy-resolution.md`
2. `references/routes/learning-material/note-authoring.md`
3. `references/routes/learning-material/quality-gates.md`

## Route Rule

This route is intentionally local-rule driven. Locate the active learning-material tree and read its current operating and metadata rules at runtime. Do not preserve a private taxonomy snapshot inside this skill.

## Output

Write a reusable note, not a transcript. Preserve the conclusion, mechanism, evidence, limits, and source links.
